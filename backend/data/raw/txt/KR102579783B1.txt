=== ABSTRACT ===
A vision inspection system and method through remote learning of product defect images are disclosed. The vision inspection system through remote learning of product defect images is connected to cameras, sensors, LED lights, and controllers, and is equipped with a machine vision inspection system equipped with machine vision image analysis software, reads the product ID, and reads the product's camera image data. An edge platform including a defect inspection module that provides shape judgment inspection, foreign matter inspection, or scratch inspection, a defect judgment module, and a manufacturing data transmission module for learning; Middleware connected to the edge platform and interoperable with the service platform; And it is linked with the edge platform through the middleware, provides defect judgment manufacturing intelligence and defect prediction manufacturing intelligence, detects atypical defective images by comparing them with accumulated and stored defect image learning data, and detects the presence or absence of foreign substances in the camera image data of the product. It provides inspection, shape inspection, and normal/defective judgment result data, and includes a service platform that provides vision inspection through remote learning of product defect images using deep learning algorithms.

  제8항에 있어서, 상기 머신 비전 영상 분석 SW는 i) 그레이 스케일 영상 처리를 사용하며, 카메라 영상 데이터(RGB 영상)을 그레이 스케일 영상 데이터로 변환하고, 그레이 스케일 영상을 사용하여 버퍼링하여 저장하고, 이미지 프로세싱 및 영상 분석 기능을 제공하며, 관심 영역(ROI, Region of Interest)을 그레이 스케일(grayscale)로 변환하며, 관심 영역(ROI)의 외곽선이 추출되는 실험에 의한 특정 임계값(threshold)을 사용하여 관심 영역(ROI)의 이미지의 히스토그램(histogram)을 구하고, 상기 이미지의 히스토그램은 x축에 영상의 각 픽셀의 화소 값, y축에 해당 화소 값의 갯수(빈도수)를 표시하며, 그 관심영역 이미지를 Otsu 알고리즘을 사용하여 임계값을 기준으로 0,1로 이진화(binarization)한 후, 히스토그램 평활화(histogram equalization)를 통해 관심 영역(ROI)의 영상의 전처리를 수행하며, sobel edge operator(소벨 엣지 연산자, sobel mask) 또는 canny edge operator(캐니 엣지 연산자)를 사용하여 x 방향의 미분과 y 방향의 미분을 구하고, sobel mask의 가중치를 영상의 화소 값에 곱하고 이들 더하는 convolution에 의해 관심 영역 이미지의 엣지(edge, 물체 영역, 배경 영역의 경계에 위치하는 화소들)를 검출하고 특정 임계값을 적용하여 생성된 엣지 영상의 결함 객체들의 윤곽선을 검출하고, 형상 특징(Shape Features)을 추출하고,상기 Otsu 알고리즘을 사용한 임계 방법을 사용하는 경우, 각각의 화소 값들이 특정 임계값을 기준으로 입력 영상에 대하여 물체 영역과 배경 영역으로 분리하고, 화소 값이 특정 임계값 보다 클 경우 물체 영역에 속하는 화소로 결정하며, 반대로 화소 값이 특정 임계값 보다 작은 경우 배경 영역에 속하는 화소로 결정하는, 제품 결함 이미지 원격 학습을 통한 비전 검사 시스템.According to clause 8, The machine vision image analysis SW uses i) gray-scale image processing, converts camera image data (RGB image) into gray-scale image data, buffers and stores using the gray-scale image, and provides image processing and image analysis functions. Provides a region of interest (ROI) that is converted to grayscale, and the outline of the region of interest (ROI) is extracted using a specific threshold based on an experiment. Obtain a histogram of the image, and the histogram of the image displays the pixel value of each pixel of the image on the After binarization to 0 and 1 based on the threshold, preprocessing of the image of the region of interest (ROI) is performed through histogram equalization, and sobel edge operator (sobel mask) or Using the canny edge operator, the differential in the x-direction and the differential in the y-direction are obtained, the weight of the sobel mask is multiplied by the pixel value of the image, and the edge (object area, pixels located on the border of the background area), detect the outlines of defective objects in the edge image generated by applying a specific threshold, and extract shape features. When using the thresholding method using the Otsu algorithm, each pixel value separates the input image into an object area and a background area based on a specific threshold, and if the pixel value is greater than a specific threshold, the pixel belongs to the object area. Conversely, if the pixel value is smaller than a certain threshold, it is determined as a pixel belonging to the background area. Image analysis SW converts camera image data (RGB image) into gray scale image data, buffers and stores the gray scale image, provides image processing and image analysis functions, and provides region of interest (ROI). Region of Interest is converted to grayscale, and a histogram [x Obtain the pixel value of each pixel of the axial image, the number (frequency) of the corresponding pixel value on the y-axis, binarize the region of interest image to 0, 1 based on the threshold using the Otsu algorithm, and then create a histogram. Preprocessing of the image of the region of interest (ROI) is performed through histogram equalization, and differentiation in the x-direction and y-direction is performed using the sobel edge operator (sobel mask) or canny edge operator (canny edge operator). Find the derivative of , multiply the weight of the sobel mask by the pixel value of the image, and detect the edge (pixels located at the border of the object area and background area) of the image of the area of interest by convolution adding them, and set a specific threshold. By applying this method, the outlines of defective objects in the generated edge image are detected and shape features are extracted.
