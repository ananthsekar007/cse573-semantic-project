=== ABSTRACT ===
The invention discloses a network channel security protection system for power 5G applications. The system includes a cloud layer and a management layer. The cloud layer includes a production control area, an information management area, an Internet area and a 5G operation core network, and the management layer includes power dispatching data. network, data communication network and operator network; in the process of network channel transmission of business data between the cloud layer and the management layer, anti-interference protection is applied to the business data through the low-latency business channel anti-interference technology to avoid the low-latency business channel. Abnormal transmission due to interference; network channel isolation technology is used to protect the power dispatching data network, data communication network and operator network to meet the access security and operation security of power services; 5G operation core network and operator network The network channel isolation technology is used for maintenance, and both the 5G operation core network and the operator network use the slice isolation technology to isolate the interacting network slices to prevent unauthorized access between network slices.

=== CLAIMS ===
  1. A network channel safety protection system for electric power 5G application is characterized by comprising: the cloud layer comprises a production control area, a management information area, an Internet area and a 5G operation core network, and the management layer comprises a power dispatching data network, a data communication network and an operator network;
      in the process of transmitting the service data between the cloud layer and the management layer through a network channel, performing anti-interference protection on the service data through a low-delay service channel anti-interference technology;
      the power dispatching data network, the data communication network and the operator network are protected by a network channel isolation technology;
      the 5G operation core network and the operator network are maintained through a network channel isolation technology, and the 5G operation core network and the operator network both isolate the interactive network slice through a slice isolation technology.
  2. The electric power 5G application-oriented network channel safety protection system according to claim 1, wherein data transmitted in the network channel is protected between the production control area and the management information area and between the management information area and the Internet area through a network channel isolation technology.
  3. The network channel safety protection system facing to the electric power 5G application as claimed in claim 1, wherein the service data of the production control area is communicated with the electric power dispatching data network through a network channel, and the management information area and the internet area are communicated with the data communication network through the network channel.
  4. The power 5G application-oriented network channel safety protection system as claimed in claim 1, wherein the data communication network comprises a management information area core network slicing service and an internet area core network slicing service, and data transmission between the management information area core network slicing service and the internet area core network slicing service is subjected to safety protection through a slicing isolation technology.
  5. The network channel security protection system for power 5G applications according to claim 4,
      the management information area core network slicing service establishes communication with the management information area through a network channel;
      and the core network slicing service of the Internet area establishes communication with the Internet area through a network channel.
  6. The power 5G application oriented network channel security protection system of claim 1, wherein the low latency traffic channel immunity techniques include immunity coding techniques, immunity high availability techniques, and interference source location techniques, wherein,
      the anti-interference coding technology obtains a pre-coding matrix through a 5G millimeter wave large-scale MIMO mixed pre-coding algorithm and a bird group algorithm, and protects the service in the transmission process of a low-delay service channel;
      the anti-interference high-availability technology is used for safe and quick channel switching so as to keep network connection uninterrupted when the mobile terminal moves from the first access network to the second access network;
      the interference source positioning technology determines a positioning estimation value through multi-device cooperative positioning and two-layer data fusion.
  7. The power 5G application oriented network channel security protection system of claim 1, wherein the network channel isolation techniques include terminal access security isolation, network domain security isolation, extranet device access security isolation, wherein,
      the terminal access security isolation authenticates the access terminal by obtaining the terminal identity authentication information to determine the validity of the access terminal;
      the network domain safety isolation prevents access and malicious invasion from a non-service flow slice by authenticating and authorizing the service flow slice;
      the access security isolation of the external network equipment is realized by setting a security access area, so that the security interaction of service flow slices is guaranteed.
  8. The power 5G application oriented network channel safeguard system according to claim 1, wherein the slice isolation technique comprises carrier isolation and RB resource reservation, wherein,
      carrier isolation distinguishes resources among network slices by using air interface resources of corresponding carrier cells through the slices;
      and the RB resource reservation is carried out by allocating the RB resources of the carrier cell to each network slice through a resource reservation mechanism.
  9. The power 5G application oriented network channel safeguard system according to claim 8, wherein the RB resource reservation comprises a static reservation and a dynamic sharing, wherein,
      the static reservation is that the resources correspondingly reserved for the specified network slice are fixed resources, and other network slices cannot be allocated;
      and dynamically sharing the resources correspondingly reserved for the specified network slice, and allocating the resources to other slices for multiplexing.
  10. The power 5G application oriented network channel safeguard system according to claim 1, wherein the slice isolation technique further comprises:
      the isolation of the interactive network slice between the data communication network and the operator network is realized through a soft isolation technology or a hard isolation technology;
      the interactive network slices are isolated by the 5G operating core network.
