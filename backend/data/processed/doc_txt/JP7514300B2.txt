 An apparatus using a model trained using artificial intelligence, comprising: The apparatus comprises one or more processors; The one or more processors: acquiring or receiving angiographic image data; receiving or loading a trained model from memory, the trained model having been trained using artificial intelligence with training data including at least angiographic frames of angiographic image data to at least detect marker locations on the angiographic frames of the acquired or received angiographic image data; applying the trained model to the acquired or received angiographic image data; selecting an angiographic frame of the one or more angiographic frames of the acquired or received angiographic image data after the trained model has been applied; detecting marker locations on the selected angiography frames using the trained model; (i) verifying whether the marker location is correct using output data from the trained model if the training data of the trained model further includes actual marker location data, and/or (ii) using input from a user of the device, wherein the output data of the trained model or the input from the user indicates whether the marker location is correct or indicates a correction to be made to the marker location if the marker location is not correct; if the marker location was not correct , modifying the detected marker location using the output data from the training model indicating the modification to be made to the marker location and/or using the input from the user of the device indicating the modification to be made to the marker location and repeating the checking whether the modified marker location is correct or, if the marker location is correct , checking whether all angiography frames of the one or more angiography frames have been checked for containing a correct marker location ; if all of the angiography frames have not been checked for containing a correct marker location , selecting another angiography frame of the one or more angiography frames and repeating the marker location detection and checking for the correct marker location for the other angiography frame; An apparatus that functions to perform the above .
