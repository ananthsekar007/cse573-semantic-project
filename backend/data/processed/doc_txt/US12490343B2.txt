5G/4G/3G/2G cloud-native OpenRAN architecture. Systems, method sand computer readable medium are provided for proving a cloud-native OpenRan architecture. In one embodiment a system includes a 5G/4G/3G/2G cloud-native Open Radio Access Network (RAN) architecture open and standardized across multiple domains; wherein the multiple domains include at least one of RAN, edge, core, orchestration and analytics; and wherein the system includes an EPC virtual stack; a Radio Virtualization stack; and an Open RAN orchestrator in communication with the EPC virtual stack and the Radio Virtualization stack, wherein the Open Ran orchestrator provides communication between any haul in communication with the EPC virtual stack and any core in communication with the Radio Virtualization stack. 1. A wireless communication system comprising:
a core-network virtualization stack in communication with multiple radio access nodes that operate according to respective radio access technologies, wherein the core-network virtualization stack virtualizes multiple core networks into resources for the multiple radio access nodes, and wherein each radio access node defines at least one respective cell; a radio virtualization stack having a virtual baseband unit (vBBU) and being in communication with the multiple core networks, wherein the radio virtualization stack virtualizes the multiple radio access nodes into dynamically-allocable resources; and an orchestrator in communication with the multiple radio access nodes via the core-network virtualization stack and in communication with the multiple core networks via the radio virtualization stack, wherein the orchestrator facilitates communication, via the core-network virtualization stack and the radio virtualization stack, between the radio access nodes and the core networks.
