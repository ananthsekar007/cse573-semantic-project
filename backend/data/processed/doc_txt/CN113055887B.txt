Network channel safety protection system for electric power 5G application. The invention discloses a network channel security protection system for power 5G applications. The system includes a cloud layer and a management layer. The cloud layer includes a production control area, an information management area, an Internet area and a 5G operation core network, and the management layer includes power dispatching data. network, data communication network and operator network; in the process of network channel transmission of business data between the cloud layer and the management layer, anti-interference protection is applied to the business data through the low-latency business channel anti-interference technology to avoid the low-latency business channel. Abnormal transmission due to interference; network channel isolation technology is used to protect the power dispatching data network, data communication network and operator network to meet the access security and operation security of power services; 5G operation core network and operator network The network channel isolation technology is used for maintenance, and both the 5G operation core network and the operator network use the slice isolation technology to isolate the interacting network slices to prevent unauthorized access between network slices. 1. A network channel safety protection system for electric power 5G application is characterized by comprising: the cloud layer comprises a production control area, a management information area, an Internet area and a 5G operation core network, and the management layer comprises a power dispatching data network, a data communication network and an operator network;
 in the process of transmitting the service data between the cloud layer and the management layer through a network channel, performing anti-interference protection on the service data through a low-delay service channel anti-interference technology;
 the power dispatching data network, the data communication network and the operator network are protected by a network channel isolation technology;
 the 5G operation core network and the operator network are maintained through a network channel isolation technology, and the 5G operation core network and the operator network both isolate the interactive network slice through a slice isolation technology.
